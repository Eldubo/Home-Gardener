## Run Locally

Clone the project

```bash
  git clone https://github.com/Eldubo/Home-Gardener.git
```

Go to the project directory

```bash
  cd Home-Gardener
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npx expo start --tunnel
```
